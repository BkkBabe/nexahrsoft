📘 Self-Help Group (SHG) Contributions — Developer Specification
Jurisdiction: Singapore Regulatory Framework: MOM + CPF Board Processing Method: Submitted together with CPF contributions

1️⃣ Overview
Singapore requires employers to deduct Self-Help Group (SHG) contributions based on an employee’s:
* Ethnicity OR Religion
* Residency Status (Singaporean vs Foreigner)
* Monthly Total Wages
The four SHGs are:
Fund	Applies To	Entity Reference
CDAC	Chinese ethnicity	Chinese Development Assistance Council
SINDA	Indian ethnicity	Singapore Indian Development Association
MBMF	Muslim employees (religion-based)	Mosque Building and Mendaki Fund
ECF	Eurasian ethnicity	Eurasian Community Fund
2️⃣ Definitions
2.1 Residency Status
For system purposes:
Singaporean = Singapore Citizen OR Permanent Resident (PR)
Foreigner = All Work Pass holders (EP, S Pass, WP, etc.)

2.2 Monthly Total Wages (MTW)
Use CPF definition:
MTW = Ordinary Wages + Additional Wages
Includes:
* Basic salary
* Fixed allowances
* Variable bonuses (in month paid)

3️⃣ Contribution Determination Logic
3.1 Core Flow
IF employee.religion == "Muslim"
    APPLY MBMF
ELSE
    IF employee.ethnicity == "Chinese"
        APPLY CDAC
    ELSE IF employee.ethnicity == "Indian"
        APPLY SINDA
    ELSE IF employee.ethnicity == "Eurasian"
        APPLY ECF
    ELSE
        NO SHG CONTRIBUTION
⚠ MBMF overrides ethnicity if employee is Muslim.

4️⃣ Residency Rules (Singaporean vs Foreigner)
Fund	Singaporean	Foreigner
CDAC	Yes	No
SINDA	Yes	Yes (Indian descent)
ECF	Yes	No
MBMF	Yes	Yes (All Muslims)

6️⃣ Opt-Out Rules
Fund	Opt-Out Allowed
CDAC	Yes
SINDA	Yes
ECF	Yes
MBMF	No (generally mandatory for Muslims)
System Logic:
IF fund != MBMF AND employee.opt_out == TRUE
    contribution = 0

7️⃣ Mixed Ethnicity Handling
If multiple ethnicities declared:
* Use primary ethnicity in official record.
* If religion = Muslim → MBMF applies regardless.

8️⃣ Payroll Requirements
* Deduct monthly.
* Submit together with CPF.
* Employee-funded only (no employer matching).
* Not part of CPF savings.

9️⃣ Payslip Display Format
SHG - CDAC     $X.XX
SHG - SINDA    $X.XX
SHG - MBMF     $X.XX
SHG - ECF      $X.XX
Only one SHG should apply per employee.
